Block and Inline Element :

In html, all the tags are categorized into two parts :

  • Block tag
  • Inline tag

Block Tag

  • block tag always starts from a new line
  • block tag takes the full width from extreme left to extreme right

Inline Tag

  • Inline tag does not starts from new line if the space is available.
  • inline tag takes the space according to its content.
